# Music Salon Manager

## Description
Консольное приложение для управления музыкальным салоном. Позволяет вести учёт компакт-дисков, их поступлений и продаж. Реализована ролевая модель (менеджер/покупатель), генерация отчётов (остатки, популярные CD/исполнители, статистика по авторам), автоматическая проверка остатка при продаже.

## Installation

### Requirements
- GCC or Clang
- SQLite3
- Make

### Setup

git clone https://github.com/Krutiwechka/tpmp-lab4.git
cd tpmp-lab4
make
./bin/music_salon

## Usage

### Authentication

| Роль | Логин | Пароль | Доступ |
|------|-------|--------|--------|
| Менеджер | manager | 1234 | Все функции |
| Покупатель | customer | 5678 | Только отчёты (*) |

### Main functions
- Управление CD (добавление, изменение цены, удаление)
- Регистрация поступлений и продаж
- Отчёты: остатки CD, продажи за период, самый популярный CD (*), самый популярный исполнитель (*), статистика по авторам
